# ReminderProject

ReminderProject is a program that allows you to record scheduled events.

The program was created in QT, the data is stored in a SQLite database.

Basic functions of the program:
- Logging with MD5 encryption
- Adding items
- Basic element validation 
- Sorting in several categories 
- Navigating through records
- Deleting a record with confirmation
- Deleting all records with confirmation
- Searching 

Some sorting is done on the database side. Database scripting is also used for data validity.

The graphic comes from gradienta.io, and is completely free. The graphic was customized and the logo created in Adobe Photoshop.
